Инструкция

1 Пригласите бота
Пригласите бота AiryPay на ваш Discord сервер.
2 Добавьте роли
Добавьте на сервер роли, которые будете продавать.
3 Добавьте товары
Добавьте товары для продажи при помощи команды /product add.

Создайте сообщение для продажи при помощи команды /setup.
